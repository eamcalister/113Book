One way to describe this process is
S1:
Add \(1\) to \(x\text{,}\)
S2:
divide \(2\) by S1.
Note that we need to be careful in describing S2; we need to specify which quantity is divided by which here. We may want to think of this as a process which is being done primarily to \(x\text{,}\) and break it down as \(2 \cdot \frac{1}{x+1}\text{.}\) Then we could describe the expression by the following:
S1:
Add \(1\) to \(x\)
S2:
Take the reciprocal of S1
S3:
multiply the result of S2 by \(2\text{.}\)
