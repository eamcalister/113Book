Fundamental Properties 1.3.2. Properties of Exponents:.

  1. \(a^na^m = a^{n+m}\text{.}\) For example: \(2^{3}\cdot 2^4 = 2\cdot 2\cdot 2\cdot 2\cdot 2\cdot 2\cdot 2 = 2^7 = 128\text{.}\)
  2. If \(a\neq 0\text{,}\) \(\frac{a^n}{a^m} = a^{n-m}\text{.}\) For example: \(\frac{3^{5}}{3^2} = \frac{3\cdot 3\cdot 3\cdot 3\cdot 3}{3\cdot 3} = 3^3 = 27\text{.}\)
  3. \((a^n)^m = a^{nm}\text{.}\) For example: \((2^{3})^2 = (2\cdot 2\cdot 2)\cdot(2\cdot 2\cdot 2) = 2^6 = 64\text{.}\)
  4. \((ab)^n = a^nb^n\text{.}\) For example: \((2\cdot 3)^2 = (2\cdot 3)\cdot (2\cdot 3) = 2\cdot 2\cdot 3\cdot 3\) \(= 2^2\cdot3^2 = 36\text{.}\)
  5. \(\left(\frac{a}{b}\right)^{n} = \frac{a^n}{b^n}\text{.}\) For example: \(\left(\frac{2}{3}\right)^2 = \frac{2}{3}\cdot\frac{2}{3} = \frac{4}{9}\text{.}\)
