Fundamental Properties 1.2.5.

Fundamental Fraction Properties:
There are four fundamental properties that we must internalize about fraction in order to do arithmetic with them:
  1. \(\displaystyle \dfrac{a}{a} = 1\)
  2. \(\displaystyle \dfrac{a}{1} = a\)
  3. \(\displaystyle x\cdot\dfrac{a}{b} = \dfrac{xa}{b}\)
  4. \(\displaystyle \dfrac{xa}{xb} = \dfrac{a}{b}\)
