Solution 5.2.13.1.

We see \(a=5\text{,}\) thus we must have
\begin{align*} 5x^2-15x+11 \amp = \amp 5(x-h)^2+k\\ \amp = \amp 5(x^2-2hx+h^2) + k\\ \amp = \amp 5x^2 - 10hx + 5h^2 + k. \end{align*}
Now we match like terms on either side of the equation to obtain the system of equations
\begin{align*} -10h \amp = \amp -15\ \text{(from the } x \text{ terms)}\\ 5h^2 + k\amp = \amp 11\ \text{(from the constant terms)} \end{align*}
Solving this system we have \(h = \frac{3}{2}\) and \(k = -\frac{1}{4}\text{.}\) As in the last example, we can solve
\begin{equation*} 5\left(x-\frac{3}{2}\right)^2 - \frac{1}{4} = 0 \end{equation*}
in a straightforward way to get \(x = \frac{3}{2}\pm\frac{1}{\sqrt{20}} = \frac{3}{2}\pm\frac{1}{2\sqrt{5}}\text{.}\)
