Solve the following equations for the indicated variable.

  1. \((3x-7)^3 = 8\text{.}\) Solve for \(x\text{.}\)

  2. \(7+2x = 2+3(x+1)\text{.}\) Solve for \(x\text{.}\)

  3. \(1-(x-7) = 2x+4\text{.}\) Solve for \(x\text{.}\)

  4. \(4(t-1)^2 = 16\text{.}\) Solve for \(t\text{.}\)

  5. \(\frac{2p-1}{3-4p} = 8\text{.}\) Solve for \(p\text{.}\)

  6. \(\frac{1}{x} + \frac{2}{x+1} = 0\text{.}\) Solve for \(x\text{.}\)

  7. \(\frac{1}{2x+7} = \frac{3}{x-1}\text{.}\) Solve for \(x\text{.}\)

  8. \(|2x-1| = 5\text{.}\) Solve for \(x\text{.}\)
